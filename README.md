 If you worked with the code from the video, please now update the margins, etc., to the margins etc. from my code.
 If the layout already looks fine, then you need to change nothing (mine was really off, so I tried a lot of things and changed a lot of things).



 We are now going to add the JS. Because I am not so good with JS, I used this one: https://rentry.org/scripted-change-floating-hearts-background.
 This JS makes the hearts “float” on the whole screen, so we need to change that. To do so, we are going to add a zIndex.

 Add this line of code in the block at the beginning of the JS  where a lot of things start with dvs. It should be directly under the part with the variabels.
  Set the zIndex to -1. 
 After that we are going to change the hearts to a symbol of our choice. You can use a lot or even text; just type it in between the two symbols.
 You can change the color too.
 I recommend a color that is near your most-used color on your side, so for example. I used pink a lot, so I checked the colors on the color site I used (there are soooo many; just use one of your choice), and when you find the color, use a light version of it. For me, that was a peachy tone. It is important to choose the right color so we keep the focus on the big container (or whatever you want to call the big white thing with the frame) and not on the animation.
